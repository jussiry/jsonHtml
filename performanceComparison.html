Performance comparison

Performance comparison can be highly misleading, since they warry so much between the parsed template and inpu data. But here's an attempt anyway.

first - HTML string

For a baseline, there is a template created as a string and inserted to elements 'innerHTML'

second - jsonHtml

Same HTML tree in object syntax

  'div#foo':
    'button.bar':
      a:
        href:"http://jussir.net/#/edit/jsonHtmlTest"
        text: 'self reference'
    div: "foo bar baz"
    ul:
      for i in [1,2]
        li: i

third - CoffeeKup

And finally in function syntax. CoffeeKup is precompiled to template - without precompiling it would be about 25 times slower.

  div '#foo', ->
    button '.bar', ->
      a
        href:"http://jussir.net/#/edit/jsonHtmlTest",
        'self reference'
    div "foo bar baz"
    ul ->
      for i in [1,2]
        li(i)
